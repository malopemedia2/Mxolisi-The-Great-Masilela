# Mxolisi “The Great” — One-page site


This is a minimal informational site scaffold (Vite + React + Tailwind) showcasing Mxolisi “The Great” Masilela and the TX Theatre documentary materials.

Getting started:

```bash
npm install
npm run dev
```

Notes:
- This is an informational site only: there are no trailers, no booking forms, and no client-side submission handlers.
- The site uses a single static hero image at `public/assets/portrait.svg`.
- The headline and portrait link directly to the full profile information.
- Tailwind is configured in `tailwind.config.cjs`.

Design highlights:
- Sparkling gold H2 headings using the `sparkle` class.
- Dark, theatrical palette and spotlight gold accents.
